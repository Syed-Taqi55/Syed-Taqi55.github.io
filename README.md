# Syed Taqi - Portfolio Website

Welcome to my personal portfolio website! This is a modern, responsive website showcasing my skills, experience, and projects.

## Features

- 🎨 Modern and responsive design
- 📱 Mobile-friendly layout
- 🚀 Smooth scrolling navigation
- 💼 Experience timeline
- 🛠️ Skills showcase
- 📁 Project portfolio
- 📧 Contact form

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons

## Getting Started

1. Clone this repository:
```bash
git clone https://github.com/Syed-Taqi55/Syed-Taqi55.github.io.git
```

2. Open `index.html` in your web browser
